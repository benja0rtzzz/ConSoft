Tipos de datos en JavaScript

  1. String:

    2. Un string en JavaScript es una secuencia de caracteres usada para representar texto.

  2. Número:

    3. Es un tipo de datos numérico, utiliza el formato en coma flotante de doble precisión, ocupa 64 bits en la memoria y ayuda a representar un amplio rango de valores.

  3. Boolean:

    4. Es un tipo de dato lógico que solo puede tener los valores true or false.

  4. Indefinido:

    5. Corresponte a un valor primitivo asignado de forma automática a las variables que solo han sido declarados para las cuales no existe argumentos reales.

  5. BigInt:

    6. Es un tipo de dato numerico que puede representar números enteros en el formato de precision arbitrario

  6. Symbol:

    7. Es un objeto incorporado cuyo constructor devuelve un symbol primitivo. Los Symbols se utilizan a menudo para añadir claves de propiedades únicas a un objeto.

  7. Null:

    8. Un valor null representa una referencia que apunta, generalmente intencionadamente, a una inexistente o inválido objeto o dirección.

  8. Object:

    9. El object se refiere a una estructura de datos que contiene datos e instrucciones para trabajar con los datos.

  9. Function:

    10. Una función es un fragmento de código que puede ser llamado por otro código o por sí mismo, o por una variable que haga referencia a la función.
